# 🛡️ SOC Analyst Lab Repository

This repository contains hands-on Security Operations Center (SOC) exercises focused on detection, analysis, threat intelligence, incident response, and attack simulation in a controlled lab environment.

## 🔎 Scope
The lab demonstrates real-world SOC analyst responsibilities including monitoring, investigation, escalation, and response using industry-standard security tools.

## 📂 Repository Structure
- SOC detection and alert analysis  
- Threat intelligence integration  
- Incident investigation and escalation  
- Evidence collection and preservation  
- Attack simulation and detection  
- End-to-end SOC workflow  

## 🛠️ Technologies Used
Wazuh SIEM · TheHive · Kali Linux · Metasploit · Velociraptor · AlienVault OTX · VirusTotal · Windows Endpoint · Metasploitable

## 🎯 Key Capabilities Demonstrated
- Log analysis and correlation  
- Anomaly detection  
- Alert triage and investigation  
- Threat hunting using MITRE ATT&CK  
- Incident escalation workflow  
- Evidence preservation and chain of custody  
- Attack detection and containment  
- SOC incident response lifecycle  

## 📄 Documentation
Each folder contains detailed reports and supporting screenshots for the performed SOC tasks.

## 📌 Purpose
To simulate practical SOC operations and demonstrate detection, investigation, and response skills in a structured security monitoring environment.
